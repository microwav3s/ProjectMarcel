Documentation IR_P01 ProjectMarcel

Our implementation utilizes the Apache Lucene Java Library for most of the IR related functions for the given task including analyzing the given documents (tokenization) and ranking.
File input is mostly handled by the jsoup HTML manipulation library for Java.

According to the task we are using the Lucene english analyzer to tokenize the HTML body texts.

Our default search index weightings are as follows:
Title: 0.5
Body: 2.0
Date: 0.5

Please consult the source code documentation inside the source files for further information.
